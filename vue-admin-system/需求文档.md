# 当前任务
1、实现供应商管理模块，提供可视化界面交互，包括：供应商的查询和列表展示、新增供应商、编辑供应商、供应商详情、添加合同。

- 供应商的基础字段：Provider Name、Country、Notes
- 供应商名下可以关联多个合同，列表中点击“添加”图标可以弹框创建新的合同。
- 供应商详情页面中需要展示供应商基本信息、名下关联的多个合同、供应商基本信息的修改记录。

2、实现合同管理模块，提供可视化界面交互，包括：合同的查询和列表展示、新增合同、编辑合同、合同详情、上传合同附件、下载合同附件。

- 合同的基础字段：Contract  Name、 Start Date、End Date、 Provider、Commercial Rights、**Notes
- 合同归属于某个供应商，编辑时不可修改。
- 支持的平台有：IPTV、Mobile、SmartTV、Web（来源于数据字典），每个平台是否具有Commercial Rights支持人工设置。
- 合同名下可以关联多个许可证，建立关联的操作后续补充。
- 合同详情页面中需要展示合同基本信息、名下关联的多个许可证、合同基本信息的修改记录。

3、实现许可证管理模块，提供可视化界面交互，包括：许可证的查询和列表展示、新增许可证、编辑许可证、许可证详情。

- 许可证的基础字段：License  Name、Contract、Start Date、 End Date、Service Type、Regions、Platform、Regions、Notes、Advertorial Rights、Downloadable for Mobile、Preview for Mobile
- 许可证属于某个合同，编辑时不可修改。
- 许可证Platform候选项有：IPTV、Mobile、SmartTV、Web（来源于同一个数据字典），且选择了几个平台界面才会显示几个平台的****Advertorial Rights供**人工设置
- Downloadable for Mobile支持人工设置开关，仅开启时，显示Download Duration (Days)供人工录入数字
- Preview for Mobile支持人工设置开关，仅开启时，显示Begin Time和End Time供人工选择时间，且必填。
- 供应商详情页面中需要展示供应商基本信息、名下关联的多个合同、供应商基本信息的修改记录。
